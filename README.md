#### Варианты запуска приложения   
- Открыть проект в IntelliJ Idea, запустить конфигурацию **Run full app**
- Докером:
  1. В линуксе из корневой папки проекта `./run.sh`
  2. В виндовсе из корневой папки проекта `run.bat`
